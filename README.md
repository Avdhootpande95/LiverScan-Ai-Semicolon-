Why LiverScan AI
  Reduces false tumor alarms
  Uses imaging and blood reports together
  Avoids overconfident predictions
  Explains results in simple, patient-friendly language

Key Features
  Multi-agent AI decision flow
  CT/MRI liver scan analysis
  Liver Function Test (LFT) correlation
  Confidence calibration & uncertainty handling
  Patient support chatbot

High-Level Flow
  Upload CT/MRI scan and blood report
  Imaging and lab analysis
  Multi-factor validation
  Final result: Tumor / Normal / Uncertain
  Simple explanation via chatbot

Tech Stack
  Google Gemini API
  Firebase & Cloud Functions
  Google Cloud Storage
